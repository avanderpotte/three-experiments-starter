# three-experiments-starter
3D Experiments starter to test cool things fast :rocket:

### install : 
__yarn__
-> install dependencies
### dev :
__yarn run dev__
-> watch files for dev
### prod :
__yarn run build__
-> build in dist/ directory

## Technologies

* Three.js
* Stylus
* ES6 w/ Babel
* Webpack 2
* StandardJs linting with eslint
* Postprocessing
* ControlKit
* Orbit controls
